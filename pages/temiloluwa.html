Exploratory Data Analysis (EDA) & Preprocessing

My analysis began by cleaning and structuring the CASAS Aruba and Milan datasets. These datasets include motion, door, and temperature sensors each producing irregular, event-driven readings. To prepare them for modeling, I:

  • Removed duplicate activations and corrected chronological inconsistencies.
  • Encoded categorical sensor states numerically (ON/OFF ‚Üí 1/‚àí1) and normalized temperature readings.
  • Handled missing sensor readings using forward imputation to maintain continuity across time steps.

Since sensors activate at irregular intervals, I engineered new temporal features: Œîtglobal (time since last event) and Œîtsensor (time since sensor activation), both log-normalized to handle scale skew. This significantly improved model performance over fixed sampling.

Pipeline Development

I designed a complete machine learning pipeline, from temporal feature extraction to model evaluation, to benchmark traditional and graph-based architectures for Human Activity Recognition (HAR) and Prediction (HAP).

  • Structured the data into 4D tensors (windows √ó steps √ó sensors √ó features).
  • Implemented and optimized two models: Graph WaveNet (GWN), and Adaptive Graph Convolutional Recurrent Network (AGCRN).
  • Used 3-fold walk-forward validation to preserve temporal order and avoid data leakage.
  • Conducted hyperparameter optimization with Optuna using Bayesian optimization.

Results

Across both datasets, Spatial-Temporal Graph Neural Networks (STGNNs) achieved accuracy comparable to CNNs and LSTMs but required significantly more computation.

  • HAR: On Aruba, LSTM and AGCRN achieved the best weighted F1 (0.82). On Milan, CNN slightly outperformed others (0.54 F1).
  • HAP: Performance dropped on both datasets (‚â§0.56 F1). AGCRN again matched the best baseline.
  • Computation: AGCRN required up to 2.2 hours of training (vs. 3.5 min for LSTM) and was approximately 150√ó slower in inference.

Overall, STGNNs offered interpretability advantages, revealing inter-room activity flows ‚Äî without clear accuracy gains.

Graph Analysis

I extracted and visualized the learned adjacency matrices from AGCRN and Graph WaveNet to interpret how models captured spatial relationships in smart homes.

  • Both models learned meaningful inter-room connections rather than local co-activations.
  • AGCRN captured realistic transitions like Work ‚Üí Meal Preparation and Meal Preparation ‚Üí Relax, aligning with empirical activity probabilities.
  • These findings show that graph-based models encode semantic movement pathways, improving interpretability even when accuracy gains are modest.

This transparency strengthens model trustworthiness, a key requirement for safety-critical applications like assisted living.
